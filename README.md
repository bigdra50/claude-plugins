# Claude Code Plugins

Claude Code plugins for Unity development, Git workflow, and session management.

## Installation

```bash
# Add marketplace
claude plugin marketplace add bigdra50/claude-plugins

# Install plugins
claude plugin install unity-dev@bigdra50
claude plugin install git-workflow@bigdra50
claude plugin install session@bigdra50
```

## Plugins

- **unity-dev**: UPM release automation, Unity Editor operations
- **git-workflow**: Gitmoji commits, PR creation, release automation
- **session**: Task orchestration, state dump, branch catchup
